##### basic

```shell
[userName@hostName ~]$/#
```

- userName 表示用户名
- hostName 表示主机名
- ~ 表示当前所在目录（~用户所在的Home）
- $ 表示普通用户、#表示管理员

```shell
[程序] [选项] [参数]
```

- 程序：命令主体
- 选项：命令的可选选项，一般用 - 或者 -- 表示
- 参数：表示命令接收的参数

```shell
whoami -- 表示当前用户名
passwd -- 修改当前用户密码
su -- 切换用户
```

#### 文件操作

```shell
file -- 查看文件类型
```
